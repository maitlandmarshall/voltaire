# Yennefer — Session Slips (Scene 0)

Hand this player **one** slip at Scene 0 start. If PvP is slow, hand a second slip at end of Round 2.

## Slip — Prove You’re the Protagonist

> “Take the Crown of Ink from someone who already has it. If you end Scene 0 crowned, you start the Ludus with **a thrown satchel** (potion or Torn Page) landing at your feet.”

## Slip — Spite Letter (Loophole)

> “Shar’s perfume still lingers. If you humiliate a rival *without* dealing damage (trip them, disarm them, steal from them, expose a secret), you gain **+2 Favor**. Nonlethal. Performative. Petty. Perfect.”
