# Cromash — Session Slips (Scene 0)

Hand this player **one** slip at Scene 0 start. If PvP is slow, hand a second slip at end of Round 2.

## Slip — The Donation

> “Send Shar something she hates. During Scene 0, throw a valuable object into the Margin (it is *gone*). If you do, receive **a Key-Tag** in the smash cut.”

## Slip — Grûumsh’s Straight Line

> “Prove your strength is not borrowed. Challenge an ally to a **nonlethal** duel for the Crown (first blood or first prone). If you win, gain **+1 Favor**; if you lose, you still gain **a Gate clue**.”
