# Cornholio — Session Slips (Scene 0)

Hand this player **one** slip at Scene 0 start. If PvP is slow, hand a second slip at end of Round 2.

## Slip — Shar’s Standard

> “Demonstrate loss without cruelty. Make someone surrender an item *willingly* (even temporarily). If you succeed, Shar marks you: start the Ludus with **+2 Favor**.”

## Slip — Jealousy Is Data

> “End Scene 0 with the Crown of Ink, but do it **without dealing the most damage**. If you can, you keep **one extra item** through capture. Shar enjoys subtlety.”
