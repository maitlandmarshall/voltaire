# Dagoth — Session Slips (Scene 0)

Hand this player **one** slip at Scene 0 start. If PvP is slow, hand a second slip at end of Round 2.

## Slip — Record the Rule

> “Speak the rule aloud before you roll. If you do this **three times** in Scene 0, Voltaire gives you a private note: **one Circle/Square/Triangle solution hint**.”
