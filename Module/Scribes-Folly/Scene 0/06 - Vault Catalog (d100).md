# Vault Catalog (d100)

Roll `1d100` when the Quillmaster opens the Vault of Margins. Items are **stage-props** of Head-Space™ unless you decide otherwise.

Default pricing:
- **01–60:** 2 Key-Tags (Rare-tier)
- **61–100:** 3 Key-Tags (Legendary-tier)

If an entry doesn’t fit the moment, reroll or swap with the nearest entry in the same tier.
