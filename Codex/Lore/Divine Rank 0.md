# Divine Rank 0

#lore #divinity #rules

![Divine Rank 0 sigil](./Divine_Rank_0_sigil.png)

## Summary

“Divine Rank 0” is the threshold state between mortal and true deity: a **quasi-deific** status where belief, boons, and mythic rules begin to apply, but the being is not yet a full god.

## What it means in this campaign (as currently understood)

- **Followers matter**: worship and converted belief can become mechanically meaningful.
- **Divine logic bleeds in**: reality starts “agreeing” with the being’s narrative in small but consistent ways.
- **Not invincible**: a Rank 0 entity is still killable, limited, and subject to higher powers.

## Connection to Voltaire

- Voltaire has explicitly recorded that he is **Divine Rank 0**.
- Voltaire’s conversions/creations (fae conversions, [[Shrek]] fine print, [[Robin]] disciple) are treated as evidence of this status.
